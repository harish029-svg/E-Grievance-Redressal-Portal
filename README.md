# E-Grievance Portal

Production-quality MERN Stack complaint management platform.

## Project Structure

```
E-Grievance-Portal/
├── frontend/     # React + Vite
├── backend/      # Node.js + Express (coming soon)
└── docs/         # SRS, DFD, ER Diagram, API docs
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Tech Stack

- **Frontend:** React 19, React Router, Axios, Framer Motion, CSS
- **Backend:** Node.js, Express, MongoDB (planned)
